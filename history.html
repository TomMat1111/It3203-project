The CERN Era (1989-1991)

In 1989, Tim Berners-Lee envisioned the World Wide Web as an open platform. He designed HTTP to allow for global information sharing among scientists at CERN.

Key Milestones

  • 1990: First server (CERN httpd) created.
  • 1996: HTTP/1.0 introduces headers.
  • 1997: HTTP/1.1 becomes the web standard.
